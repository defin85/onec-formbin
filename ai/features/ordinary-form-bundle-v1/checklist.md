# Checklist

- Does the change keep the ordinary-form workspace export deterministic and raw-first?
- Does the documented bundle contract stay explicit about writable scope and unsupported areas?
- Can an agent still reach export -> edit -> apply -> pack on the verified baseline fixture?
- Do split-form workspaces remain readable while unsupported writes fail closed?
- Were public artifact names and schema/version boundaries updated in the docs when needed?
- Does the bundle stay suitable as an external ingest input without leaking private implementation details?
